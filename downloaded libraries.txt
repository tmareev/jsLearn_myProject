Если у меня есть со всеми проектами (например "MAIN"), в которой есть "node_modules" (где хранятся все бибилиотеки) 
и папка c дз (например "5 hw"), то для запуска какой-то проги любой из библиотек 
нужно перейти в папку с дз и ввести в консоль ..\node_modules\.bin\xxx 


**************************************************************************************************************
для ввода текста в ноде:
npm install readline-sync

**************************************************************************************************************
для тестов:
npm install mocha chai unexpected should

**************************************************************************************************************
сборщик и работа с ним:
npm install webpack
npm install webpack-cli
________________________________________________________________________________________________________________
собирает проект в нами созданный файл index.js (по умолчанию main.js) 
и добавили автоматическую перекомпиляцию (с помощью  --watch) 
до того момента пока мы не закроем консоль или не стопнем сам вебпак:

....5 hw>node_modules\.bin\webpack --output=dist/index.js --watch
dev запуск
 ...>node_modules\.bin\webpack-cli --mode=development

**********************************************************************
создание сервера
npm install node-static
простой запуск 
....6 hw>node_modules\.bin\static
